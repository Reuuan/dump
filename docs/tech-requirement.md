Требование к функциональным характеристикам 

 Цель игры -- угадать слово  меньше чем за 8 ходов. 

 Перед началом игры игроку предлагается выбрать уровень сложности. В начале игры программа загадывает слово из одного из своих словарей, соответствующего выбранной сложности. 
Разные словари в зависимости от сложности  содержат слова английского языка разной длинны и частотности употребления. Словари имеют текстовый формат и внутри них слова перечисляются через пробел. 
Каждый ход игрок вводит одну букву. Если в загаданном слове есть введенная буква, то она открывается на всех местах где она присутствует в слове. 
Если введенная буква отсутствует в загаданном слове, то у висилицы рисуется следующий элемент. 
Для отметки угаданных в слове букв используется вспомогательный массив типа char длины соответствующей длине загаданного слова. Изначально массив заполнен нижинми подчеркиваниями('_'),
а затем по результатам ходов подчеркивания заменяются угаданными буквами на соответствующих местах.
Игра заканчивается в двух случаях: либо игрок угадал слово не совершив 8ми ошибок, тогда он побеждает, либо игрок совершил 8 ошибок и виселица насрисована полностью,тогда он проигрывает.

Формат входных данных 

 Игрок может вводить символы латиницы любого регистра.

Интерфейс

 Запуск игры и управление ей осуществляются через терминал Linux.
